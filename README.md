# Git Commit Convention (2024-08-14)
| 태그명 | 설명 |
| --- | --- |
| Feat | 새로운 기능 추가 |
| Fix | 기능 수정 |
| Bug | 버그 수정 |
| Design | CSS 등 사용자 UI 디자인 변경 |
| Style | 코드 포맷 변경, 세미콜론 누락, 코드 수정이 없는 경우 |
| Refactor | Production Code 리팩토링 |
| Comment | 필요한 주석 추가 및 변경 |
| Docs | 문서 추가 및 수정 |
| Test | 테스트 코드 추가 및 리팩토링 |
| Chore | 빌드 업무 수정, 패키지 매니저 수정 |
| Rename | 파일 혹은 폴더명 수정 |
| Remove | 파일 삭제 |
